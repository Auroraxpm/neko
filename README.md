Depedencies
---
- Openwrt 21
  - ` php7 `
  - ` php7-cgi `
  - ` firewall `
  - ` iptables `
   
- Openwrt 22/23
  - ` php8 `
  - ` php8-cgi `
  - ` firewall4 `
  - ` kmod-tun `
  - ` xtables-nft `

How to Use
---
1. Download IPK files on Release, [here](https://github.com/nosignals/neko/releases)
2. Install on OpenWrt with Normal Connectivity (Internet)
3. Done, you can setting config on ` Configs ` tab
